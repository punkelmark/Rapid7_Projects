To install related packages, run 

pip install -r requirements.txt

Afterwards, restart the IDE to reflect changes

Make sure to be running in a virtual environment when running and importing scripts, this is to avoid conflicts when running new packages

Create venv:
py -m venv apienv

To activate:
APIENV\Scripts\Activate

In case Windows blocks script execution:
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
